# Project Management Backend

A backend system for managing projects, users, and team roles with structured API design.

## Features

- User authentication and authorization  
- Project and task management  
- Role-based access control  
- RESTful APIs for handling data  

## Tech Stack

- Node.js  
- Express.js  
- MongoDB  
- Mongoose  

## Overview

This project demonstrates a structured backend architecture using models, controllers, and routes. It focuses on building scalable and maintainable server-side applications.
